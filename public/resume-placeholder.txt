RESUME PDF INSTRUCTIONS
========================

To add your resume to the portfolio:

1. Create or export your resume as a PDF file
2. Name it exactly: resume.pdf
3. Place it in the public/ folder (same location as this file)
4. Delete this placeholder file

The resume download button in your portfolio will automatically work once you add the PDF file.

Recommended Resume Format:
- Single page or 2-page maximum
- Professional font (Arial, Calibri, Times New Roman)
- Clear sections: Education, Skills, Experience, Projects
- Include contact information and LinkedIn/GitHub links
- Export as PDF (not Word document)
- File size: Keep under 1MB for fast loading
